  --rust examples/rust_curve25519-dalek_4.1.3_atoms.json \
  --lean examples/lean_Curve25519Dalek_0.1.0.json \
  --functions examples/functions.json \
  --output /tmp/test_translations.json

## Charon name format improvement

The `{Trait for SelfType}` format in `probe-rust/charon_names.rs` resolves
the Self type from LLBC trait impls, producing names like:
```
curve25519_dalek::scalar::{core::clone::Clone for curve25519_dalek::scalar::Scalar}::clone
```

This matches the Aeneas `functions.json` format, enabling exact matching for
trait impl methods. Before this fix, only free functions and inherent impls
could be matched exactly.

| Metric | Before fix | After fix |
|---|---|---|
| Atoms with `{Trait for Type}` names | 0 | 165 |
| functions.json normalized matches | 113 | 218 |
| Strategy 1 (exact) translations | 89 | 162 |
